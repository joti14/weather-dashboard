# Weather & Air Quality Dashboard
A modern, responsive web application built with React + Vite that displays real-time weather conditions and air quality index (AQI) for any city worldwide.

## Features
- Current Weather: Temperature, feels-like, humidity, wind speed, pressure, sunrise/sunset times
- Weather Icon: Large emoji-based icon reflecting current conditions (sunny, cloudy, rain, etc.)
- Air Quality Index: Color-coded AQI (1â€“5) with progress bar and health description
- Dynamic Background: Gradient changes based on temperature 
- Recent Searches: Last 3 searched cities saved in localStorage with quick-access buttons
- Glassmorphism UI: Clean, modern card design with blur effect and subtle shadows
- Loading State & Error Handling: Smooth user feedback
  
## Technologies Used
- React (with Hooks)
- Vite 
- Axios (for API requests)
- OpenWeatherMap API (free tier)
- Vanilla CSS
